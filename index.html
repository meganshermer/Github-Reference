How to Create a Repository

Step 1. create repository

push new repository in right hand corner

Step 2. Make repository name

Make the name for the repository

Step 3. Push upload files

and then drag your files such as your html, assets, and stylesheet files.

Step 4. Commit Changes

push the commit changes button

Step 5. Home page

go back to your code page and it will show all the files that you have uploaded

Step 6. Settings

go to the settings in the up bar and scroll all the way down until you see "Github Pages"

Step 7. Master Branch

click masterbranch under the source button in the github pages tab. It will take you to the top of the page, scroll back down and there will be a blue or green link. push that.

Finished!

This will take you to your published webpage. congrats you have finished all the steps!
